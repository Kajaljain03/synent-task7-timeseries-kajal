# Time Series Analysis Project

## Objective

The objective of this project is to analyze stock price trends over time and identify important patterns using time series analysis techniques.

## Dataset

Stock Price Dataset from Kaggle

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Google Colab

## Tasks Performed

* Data preprocessing
* Date formatting
* Trend analysis
* Moving average calculation
* Monthly trend visualization

## Visualizations Created

* Stock closing price trend
* Moving average graph
* Monthly average price analysis

## Key Insights

* Stock prices fluctuated over time with noticeable trends.
* Moving averages helped identify long-term market direction.
* Monthly analysis showed variation in stock performance.

## Outcome

This project improved understanding of time series analysis, trend visualization, and stock market data interpretation using Python.
